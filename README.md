### 声明
本项目为上海交通大学软件工程课程第一次课程作业，题目如下：
>大学生经常有些物品觉得扔掉可惜，不处理又觉得浪费自己的地方。请你编写一个物品“复活”软件
>该程序允许添加物品的信息（物品名称，物品描述，联系人信息），删除物品的信息，显示物品列表，也允许查找物品的信息
>你实现的程序可以采用命令行方式使用，但是鼓励提供GUI
>程序实现完毕后，发布在Github上
>进行PSP数据的统计，发布在你的技术博客上（内容请按照下表）
